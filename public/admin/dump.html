Database dump interface

Options are:

Trips
Get data on trips made between start and end dates
Tracks
Get data on tracks made during trips between start and end dates
Trip estimates
Get estimates (distance, creels) for trips made between start and end dates
Track analysis
Get data on analysed tracks (activity, grid squares) from trips made between start and end dates
Vessels
Get data on all vessels, their owner, gear and target species (can be joined to trip data)
Grids
Get coordinates for all grid squares (can be joined to track analysis)
App creels
Get data on observed abandoned creels
App catch
Get data on catch entered into app
App observations
Get observation data from app
